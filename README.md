# 📖 Project Overview

PathWise is an AI-driven academic and career recommendation platform designed to help students discover suitable career paths based on their skills, interests, and goals.

The system uses:
- Cosine Similarity
- Weighted Scoring
- Rule-Based Filtering

to generate personalized and explainable career recommendations.

---

# 🚀 Key Features

- AI-Based Career Recommendation
- Skill Gap Analysis
- Personalized Learning Roadmaps
- Resume Parsing
- Interactive Dashboard
- Assessment & Anti-Cheating System
- Secure Authentication using JWT & OTP
- Admin Dashboard
