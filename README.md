# 📖 Project Overview

PathWise is an AI-driven academic and career recommendation platform designed to help students discover suitable career paths based on their skills, interests, and goals.

The platform analyzes user profiles and generates:
- Personalized career recommendations
- Skill gap analysis
- Structured learning roadmaps
- Explainable recommendation insights

The system uses:
- Cosine Similarity
- Weighted Scoring
- Rule-Based Filtering

to generate accurate and transparent recommendations.

# 📦 Installation

## Clone Repository

```bash
git clone <repository-url>
cd PathWise-2210990462-2210990072-2210990795
```

---

## Install Dependencies

### Root

```bash
npm install
```

### Frontend

```bash
cd frontend
npm install
```

### Backend

```bash
cd backend
npm install
```

### AI Engine

```bash
cd ai-engine
npm install
```
